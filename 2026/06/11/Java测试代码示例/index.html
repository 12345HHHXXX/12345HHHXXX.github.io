一、素数判断 (Prime Numbers)

题目描述

编写一个Java程序，判断一个数是否为素数（质数）。

素数定义：一个大于1的自然数，除了1和它自身外，不能被其他自然数整除的数。

解题思路

  1. 若数字小于等于1，则不是素数
  2. 若数字等于2，则是素数（唯一的偶素数）
  3. 若数字是偶数且大于2，则不是素数
  4. 从3开始，只检查奇数因子，直到sqrt(n)

完整代码

public class PrimeChecker {
    
    /**
     * 判断一个数是否为素数
     * @param n 待判断的整数
     * @return 如果是素数返回true，否则返回false
     */
    public static boolean isPrime(int n) {
        // 小于等于1的数不是素数
        if (n <= 1) return false;
        // 2是唯一的偶素数
        if (n == 2) return true;
        // 偶数（除了2）不是素数
        if (n % 2 == 0) return false;
        // 从3开始，只检查奇数因子，优化到sqrt(n)
        for (int i = 3; i * i <= n; i += 2) {
            if (n % i == 0) return false;
        }
        return true;
    }
    
    /**
     * 打印指定范围内的所有素数
     * @param start 起始数字
     * @param end 结束数字
     */
    public static void printPrimes(int start, int end) {
        System.out.println(start + " 到 " + end + " 之间的素数：");
        for (int i = start; i <= end; i++) {
            if (isPrime(i)) {
                System.out.print(i + " ");
            }
        }
        System.out.println();
    }
    
    public static void main(String[] args) {
        // 测试单个数字
        int num = 29;
        System.out.println(num + " 是否为素数: " + isPrime(num));
        
        // 测试边界值
        System.out.println("0 是否为素数: " + isPrime(0));
        System.out.println("1 是否为素数: " + isPrime(1));
        System.out.println("2 是否为素数: " + isPrime(2));
        System.out.println("97 是否为素数: " + isPrime(97));
        
        // 打印范围内的素数
        printPrimes(1, 50);
    }
}

97 是否为素数: true

二、枚举类型应用场景

枚举（Enum）是Java中的一种特殊类型，用于定义固定数量的常量集合。以下是枚举的三个常见应用场景：

场景1：状态/类型定义（最常用）

应用场景

在实际项目中，经常需要定义一些固定的状态，如订单状态、用户状态、支付状态等。使用枚举可以避免魔法数字，提高代码可读性和维护性。

完整代码

/**
 * 订单状态枚举
 * 包含常见的订单生命周期状态
 */
public enum OrderStatus {
    PENDING("待支付", "用户已提交订单，等待支付"),
    PAID("已支付", "用户已完成支付"),
    SHIPPED("已发货", "商家已发货"),
    DELIVERED("已送达", "订单已送达用户"),
    CANCELLED("已取消", "订单已取消"),
    REFUNDED("已退款", "订单已退款");
    
    // 状态显示名称
    private final String displayName;
    // 状态描述
    private final String description;
    
    /**
     * 构造函数
     * @param displayName 显示名称
     * @param description 状态描述
     */
    OrderStatus(String displayName, String description) {
        this.displayName = displayName;
        this.description = description;
    }
    
    public String getDisplayName() {
        return displayName;
    }
    
    public String getDescription() {
        return description;
    }
    
    /**
     * 根据显示名称获取枚举值
     * @param displayName 显示名称
     * @return 对应的枚举值，未找到返回null
     */
    public static OrderStatus fromDisplayName(String displayName) {
        for (OrderStatus status : values()) {
            if (status.displayName.equals(displayName)) {
                return status;
            }
        }
        return null;
    }
    
    public static void main(String[] args) {
        // 使用枚举
        OrderStatus status = OrderStatus.PAID;
        System.out.println("状态名称: " + status.name());
        System.out.println("显示名称: " + status.getDisplayName());
        System.out.println("状态描述: " + status.getDescription());
        System.out.println("序号: " + status.ordinal());
        
        // 遍历所有状态
        System.out.println("\n所有订单状态:");
        for (OrderStatus s : OrderStatus.values()) {
            System.out.println(s.ordinal() + ". " + s.name() + " - " + s.getDisplayName());
        }
        
        // 根据显示名称查找
        OrderStatus found = OrderStatus.fromDisplayName("已发货");
        System.out.println("\n查找结果: " + found);
    }
}

场景2：策略模式（替换大量if/else）

应用场景

当需要根据不同条件执行不同算法时，可以使用枚举实现策略模式，避免冗长的if-else语句。

完整代码

/**
 * 折扣策略枚举
 * 使用枚举实现策略模式
 */
public enum DiscountStrategy {
    
    STUDENT("学生折扣", "适用于在校学生") {
        @Override
        public double applyDiscount(double price) {
            return price * 0.8; // 学生8折
        }
    },
    
    VIP("VIP折扣", "适用于VIP会员") {
        @Override
        public double applyDiscount(double price) {
            return price * 0.7; // VIP7折
        }
    },
    
    MEMBER("会员折扣", "适用于普通会员") {
        @Override
        public double applyDiscount(double price) {
            return price * 0.85; // 会员85折
        }
    },
    
    NORMAL("无折扣", "适用于普通用户") {
        @Override
        public double applyDiscount(double price) {
            return price; // 无折扣
        }
    };
    
    private final String strategyName;
    private final String description;
    
    DiscountStrategy(String strategyName, String description) {
        this.strategyName = strategyName;
        this.description = description;
    }
    
    /**
     * 抽象方法：计算折扣后价格
     * @param price 原价
     * @return 折扣后的价格
     */
    public abstract double applyDiscount(double price);
    
    public String getStrategyName() {
        return strategyName;
    }
    
    public String getDescription() {
        return description;
    }
    
    /**
     * 计算最终价格（包含格式化）
     * @param price 原价
     * @return 格式化后的价格字符串
     */
    public String calculateFinalPrice(double price) {
        double finalPrice = applyDiscount(price);
        double saved = price - finalPrice;
        return String.format("原价: %.2f, 折扣后: %.2f, 节省: %.2f", 
                           price, finalPrice, saved);
    }
    
    public static void main(String[] args) {
        double originalPrice = 100.0;
        
        // 测试各种折扣策略
        for (DiscountStrategy strategy : DiscountStrategy.values()) {
            System.out.println("\n--- " + strategy.getStrategyName() + " ---");
            System.out.println("描述: " + strategy.getDescription());
            System.out.println(strategy.calculateFinalPrice(originalPrice));
        }
    }
}

场景3：统一返回码（后端接口必备）

/**
 * 统一返回码枚举
 * 定义后端接口的标准返回码
 */
public enum ResultCode {
    
    // 成功状态码
    SUCCESS(200, "操作成功"),
    
    // 客户端错误
    BAD_REQUEST(400, "请求参数错误"),
    UNAUTHORIZED(401, "未授权，请先登录"),
    FORBIDDEN(403, "禁止访问，权限不足"),
    NOT_FOUND(404, "资源未找到"),
    METHOD_NOT_ALLOWED(405, "请求方法不允许"),
    
    // 服务器错误
    ERROR(500, "服务器内部错误"),
    SERVICE_UNAVAILABLE(503, "服务暂不可用"),
    
    // 业务错误
    VALIDATION_ERROR(1000, "数据校验失败"),
    BUSINESS_ERROR(1001, "业务逻辑错误"),
    DUPLICATE_KEY(1002, "数据重复"),
    DATA_NOT_FOUND(1003, "数据不存在");
    
    // HTTP状态码
    private final int code;
    // 错误消息
    private final String message;
    
    ResultCode(int code, String message) {
        this.code = code;
        this.message = message;
    }
    
    public int getCode() {
        return code;
    }
    
    public String getMessage() {
        return message;
    }
    
    /**
     * 判断是否为成功状态
     */
    public boolean isSuccess() {
        return this == SUCCESS;
    }
    
    /**
     * 判断是否为客户端错误
     */
    public boolean isClientError() {
        return code >= 400 && code < 500;
    }
    
    /**
     * 判断是否为服务器错误
     */
    public boolean isServerError() {
        return code >= 500 && code < 600;
    }
    
    /**
     * 判断是否为业务错误
     */
    public boolean isBusinessError() {
        return code >= 1000;
    }
    
    public static void main(String[] args) {
        // 测试各种状态码
        ResultCode[] codes = {SUCCESS, BAD_REQUEST, ERROR, VALIDATION_ERROR};
        
        for (ResultCode code : codes) {
            System.out.println("\n--- 返回码: " + code.getCode() + " ---");
            System.out.println("消息: " + code.getMessage());
            System.out.println("是否成功: " + code.isSuccess());
            System.out.println("客户端错误: " + code.isClientError());
            System.out.println("服务器错误: " + code.isServerError());
            System.out.println("业务错误: " + code.isBusinessError());
        }
    }
}

三、匿名内部类的使用场景

题目描述

使用匿名内部类实现一个简单的计时器工具，计算方法执行的耗时。

解题思路

  1. 定义一个函数式接口（只有一个抽象方法的接口）
  2. 创建计时工具类，接收接口实现作为参数
  3. 使用匿名内部类实现接口，传入具体的业务逻辑

完整代码

/**
 * 计时器工具类
 * 使用匿名内部类实现方法耗时统计
 */
public class TimerUtil {
    
    /**
     * 任务接口（函数式接口）
     * 定义需要执行的任务
     */
    @FunctionalInterface
    public interface Task {
        /**
         * 执行任务的方法
         */
        void execute();
    }
    
    /**
     * 任务接口（带返回值）
     */
    @FunctionalInterface
    public interface TaskWithResult<T> {
        /**
         * 执行任务并返回结果
         * @return 任务执行结果
         */
        T execute();
    }
    
    /**
     * 测量任务执行时间
     * @param task 要执行的任务
     * @return 执行耗时（毫秒）
     */
    public static long measureTime(Task task) {
        long startTime = System.currentTimeMillis();
        try {
            task.execute();
        } catch (Exception e) {
            System.err.println("任务执行异常: " + e.getMessage());
        }
        long endTime = System.currentTimeMillis();
        return endTime - startTime;
    }
    
    /**
     * 测量带返回值任务的执行时间
     * @param task 要执行的任务
     * @return 包含结果和耗时的对象
     */
    public static <T> TimedResult<T> measureTimeWithResult(TaskWithResult<T> task) {
        long startTime = System.currentTimeMillis();
        T result = null;
        Exception exception = null;
        try {
            result = task.execute();
        } catch (Exception e) {
            exception = e;
        }
        long endTime = System.currentTimeMillis();
        
        return new TimedResult<>(result, endTime - startTime, exception);
    }
    
    /**
     * 带结果的计时返回对象
     */
    public static class TimedResult<T> {
        private final T result;
        private final long durationMs;
        private final Exception exception;
        
        public TimedResult(T result, long durationMs, Exception exception) {
            this.result = result;
            this.durationMs = durationMs;
            this.exception = exception;
        }
        
        public T getResult() { return result; }
        public long getDurationMs() { return durationMs; }
        public Exception getException() { return exception; }
        public boolean hasError() { return exception != null; }
    }
    
    public static void main(String[] args) {
        System.out.println("=== 测试1: 计算平方根耗时 ===");
        long duration1 = TimerUtil.measureTime(new Task() {
            @Override
            public void execute() {
                // 模拟耗时操作：计算100万次平方根
                for (int i = 0; i < 1000000; i++) {
                    Math.sqrt(i);
                }
            }
        });
        System.out.println("执行耗时: " + duration1 + " 毫秒");
        
        System.out.println("\n=== 测试2: 字符串拼接耗时 ===");
        long duration2 = TimerUtil.measureTime(new Task() {
            @Override
            public void execute() {
                String result = "";
                for (int i = 0; i < 10000; i++) {
                    result += "test" + i;
                }
            }
        });
        System.out.println("执行耗时: " + duration2 + " 毫秒");
        
        System.out.println("\n=== 测试3: 带返回值的任务 ===");
        TimedResult<Integer> result = TimerUtil.measureTimeWithResult(new TaskWithResult<Integer>() {
            @Override
            public Integer execute() {
                // 计算1到100的和
                int sum = 0;
                for (int i = 1; i <= 100; i++) {
                    sum += i;
                }
                return sum;
            }
        });
        System.out.println("计算结果: " + result.getResult());
        System.out.println("执行耗时: " + result.getDurationMs() + " 毫秒");
        System.out.println("是否有错误: " + result.hasError());
    }
}

四、Java反射机制

题目描述

使用Java反射机制获取类的信息并动态调用方法。

解题思路

  1. 通过Class.forName()获取类对象
  2. 使用反射获取类的构造器、方法、字段
  3. 通过反射创建对象实例
  4. 动态调用对象的方法

完整代码

import java.lang.reflect.Constructor;
import java.lang.reflect.Field;
import java.lang.reflect.Method;

/**
 * 反射机制演示
 * 展示如何使用反射获取类信息和调用方法
 */
public class ReflectionDemo {
    
    /**
     * 示例用户类
     */
    public static class User {
        // 私有字段
        private String name;
        private int age;
        // 公共字段
        public String email;
        
        // 构造函数
        public User() {
            this.name = "Unknown";
            this.age = 0;
        }
        
        public User(String name, int age) {
            this.name = name;
            this.age = age;
        }
        
        // Getter和Setter
        public String getName() {
            return name;
        }
        
        public void setName(String name) {
            this.name = name;
        }
        
        public int getAge() {
            return age;
        }
        
        public void setAge(int age) {
            this.age = age;
        }
        
        // 业务方法
        public void sayHello(String greeting) {
            System.out.println(greeting + ", 我是 " + name + ", 今年 " + age + " 岁");
        }
        
        public String introduce() {
            return "大家好，我是" + name + "，今年" + age + "岁";
        }
        
        // 私有方法
        private void secretMethod() {
            System.out.println("这是一个私有方法");
        }
        
        @Override
        public String toString() {
            return "User{name='" + name + "', age=" + age + ", email='" + email + "'}";
        }
    }
    
    /**
     * 打印类的详细信息
     */
    public static void printClassInfo(Class<?> clazz) {
        System.out.println("\n=== 类信息 ===");
        System.out.println("类名: " + clazz.getName());
        System.out.println("简单类名: " + clazz.getSimpleName());
        System.out.println("父类: " + clazz.getSuperclass());
        
        // 打印构造函数
        System.out.println("\n=== 构造函数 ===");
        Constructor<?>[] constructors = clazz.getDeclaredConstructors();
        for (Constructor<?> c : constructors) {
            System.out.println("构造函数: " + c);
        }
        
        // 打印方法
        System.out.println("\n=== 方法列表 ===");
        Method[] methods = clazz.getDeclaredMethods();
        for (Method m : methods) {
            System.out.println("方法: " + m.getReturnType().getSimpleName() 
                           + " " + m.getName() + "()");
        }
        
        // 打印字段
        System.out.println("\n=== 字段列表 ===");
        Field[] fields = clazz.getDeclaredFields();
        for (Field f : fields) {
            System.out.println("字段: " + f.getType().getSimpleName() + " " + f.getName());
        }
    }
    
    public static void main(String[] args) throws Exception {
        // 1. 获取类对象
        Class<User> clazz = User.class;
        
        // 2. 打印类信息
        printClassInfo(clazz);
        
        // 3. 通过反射创建对象（使用无参构造）
        System.out.println("\n=== 创建对象 ===");
        User user1 = clazz.getDeclaredConstructor().newInstance();
        System.out.println("创建的对象: " + user1);
        
        // 4. 通过反射创建对象（使用有参构造）
        Constructor<User> constructor = clazz.getDeclaredConstructor(String.class, int.class);
        User user2 = constructor.newInstance("张三", 25);
        System.out.println("创建的对象: " + user2);
        
        // 5. 通过反射调用方法
        System.out.println("\n=== 调用方法 ===");
        Method sayHelloMethod = clazz.getDeclaredMethod("sayHello", String.class);
        sayHelloMethod.invoke(user2, "你好");
        
        Method introduceMethod = clazz.getDeclaredMethod("introduce");
        String intro = (String) introduceMethod.invoke(user2);
        System.out.println("自我介绍: " + intro);
        
        // 6. 通过反射访问和修改字段
        System.out.println("\n=== 访问字段 ===");
        // 访问公共字段
        Field emailField = clazz.getField("email");
        emailField.set(user2, "zhangsan@example.com");
        System.out.println("设置后的对象: " + user2);
        
        // 访问私有字段
        Field nameField = clazz.getDeclaredField("name");
        nameField.setAccessible(true); // 突破访问限制
        String nameValue = (String) nameField.get(user2);
        System.out.println("获取私有字段name: " + nameValue);
        
        // 7. 调用私有方法
        System.out.println("\n=== 调用私有方法 ===");
        Method secretMethod = clazz.getDeclaredMethod("secretMethod");
        secretMethod.setAccessible(true);
        secretMethod.invoke(user2);
    }
}

五、文件复制竞赛题

题目描述

实现四种不同的文件复制方式，并比较它们的性能差异。

解题思路

  1. 字符流复制（适用于文本文件）
  2. 字符缓冲流复制（效率更高的文本复制）
  3. 字节流复制（适用于任意文件）
  4. 字节缓冲流复制（效率最高的通用复制）

完整代码

import java.io.*;

/**
 * 文件复制工具类
 * 包含四种不同的复制方式
 */
public class FileCopyUtils {
    
    /**
     * 方法1: 文本文件复制（字符流逐行读取）
     * 适用于文本文件，会自动处理换行符
     */
    public static void copyTextFile(String source, String dest) throws IOException {
        try (BufferedReader br = new BufferedReader(new FileReader(source));
             BufferedWriter bw = new BufferedWriter(new FileWriter(dest))) {
            String line;
            while ((line = br.readLine()) != null) {
                bw.write(line);
                bw.newLine(); // 写入换行符
            }
        }
    }
    
    /**
     * 方法2: 字符缓冲流复制（批量读取）
     * 适用于文本文件，性能优于逐行读取
     */
    public static void copyWithCharBuffer(String source, String dest) throws IOException {
        try (BufferedReader br = new BufferedReader(new FileReader(source));
             BufferedWriter bw = new BufferedWriter(new FileWriter(dest))) {
            char[] buffer = new char[8192]; // 8KB缓冲区
            int len;
            while ((len = br.read(buffer)) != -1) {
                bw.write(buffer, 0, len);
            }
        }
    }
    
    /**
     * 方法3: 字节流复制（原始字节）
     * 适用于任意文件，但性能较差
     */
    public static void copyWithByteStream(String source, String dest) throws IOException {
        try (FileInputStream fis = new FileInputStream(source);
             FileOutputStream fos = new FileOutputStream(dest)) {
            byte[] buffer = new byte[8192]; // 8KB缓冲区
            int len;
            while ((len = fis.read(buffer)) != -1) {
                fos.write(buffer, 0, len);
            }
        }
    }
    
    /**
     * 方法4: 字节缓冲流复制（推荐使用）
     * 适用于任意文件，性能最优
     */
    public static void copyWithBufferedStream(String source, String dest) throws IOException {
        try (BufferedInputStream bis = new BufferedInputStream(new FileInputStream(source));
             BufferedOutputStream bos = new BufferedOutputStream(new FileOutputStream(dest))) {
            byte[] buffer = new byte[8192]; // 8KB缓冲区
            int len;
            while ((len = bis.read(buffer)) != -1) {
                bos.write(buffer, 0, len);
            }
        }
    }
    
    /**
     * 复制文件并返回耗时
     */
    public static long copyAndMeasure(String source, String dest, CopyMethod method) {
        long startTime = System.currentTimeMillis();
        try {
            switch (method) {
                case TEXT_LINE:
                    copyTextFile(source, dest);
                    break;
                case TEXT_BUFFER:
                    copyWithCharBuffer(source, dest);
                    break;
                case BYTE_STREAM:
                    copyWithByteStream(source, dest);
                    break;
                case BYTE_BUFFER:
                    copyWithBufferedStream(source, dest);
                    break;
            }
        } catch (IOException e) {
            System.err.println("复制失败: " + e.getMessage());
            return -1;
        }
        return System.currentTimeMillis() - startTime;
    }
    
    /**
     * 复制方法枚举
     */
    public enum CopyMethod {
        TEXT_LINE("文本逐行复制"),
        TEXT_BUFFER("字符缓冲复制"),
        BYTE_STREAM("字节流复制"),
        BYTE_BUFFER("字节缓冲复制");
        
        private final String description;
        
        CopyMethod(String description) {
            this.description = description;
        }
        
        public String getDescription() {
            return description;
        }
    }
    
    /**
     * 测试性能
     */
    public static void testPerformance(String sourceFile, int testTimes) throws IOException {
        System.out.println("=== 文件复制性能测试 ===");
        System.out.println("源文件: " + sourceFile);
        System.out.println("测试次数: " + testTimes);
        
        File source = new File(sourceFile);
        if (!source.exists()) {
            System.out.println("错误: 源文件不存在!");
            return;
        }
        System.out.println("文件大小: " + source.length() + " 字节");
        
        // 测试每种方法
        for (CopyMethod method : CopyMethod.values()) {
            long totalTime = 0;
            System.out.println("\n--- " + method.getDescription() + " ---");
            
            for (int i = 0; i < testTimes; i++) {
                String destFile = "test_copy_" + method.name().toLowerCase() + ".tmp";
                long time = copyAndMeasure(sourceFile, destFile, method);
                totalTime += time;
                System.out.println("第" + (i + 1) + "次: " + time + " ms");
                
                // 删除临时文件
                new File(destFile).delete();
            }
            
            double avgTime = (double) totalTime / testTimes;
            System.out.println("平均耗时: " + String.format("%.2f", avgTime) + " ms");
        }
    }
    
    public static void main(String[] args) throws IOException {
        // 创建测试文件（如果不存在）
        String testFile = "test_large_file.txt";
        createTestFile(testFile, 100 * 1024); // 创建100KB的测试文件
        
        // 测试性能
        testPerformance(testFile, 3);
        
        // 清理测试文件
        // new File(testFile).delete();
    }
    
    /**
     * 创建测试文件
     */
    private static void createTestFile(String filename, int sizeBytes) throws IOException {
        File file = new File(filename);
        if (file.exists()) {
            System.out.println("测试文件已存在，跳过创建");
            return;
        }
        
        try (BufferedWriter bw = new BufferedWriter(new FileWriter(filename))) {
            StringBuilder sb = new StringBuilder();
            for (int i = 0; i < 1000; i++) {
                sb.append("这是第 ").append(i).append(" 行测试数据。\n");
            }
            // 重复写入直到达到目标大小
            String content = sb.toString();
            while (file.length() < sizeBytes) {
                bw.write(content);
            }
        }
        System.out.println("创建测试文件: " + filename);
    }
}

六、线程创建方式对比

题目描述

比较使用Thread子类创建线程与使用Runnable接口创建线程的区别。

核心区别

特性 Thread子类方式 Runnable接口方式
继承限制 Java单继承，无法继承其他类 可继承其他类，灵活
代码复用 较差，线程逻辑与Thread耦合 较好，任务逻辑可被多个线程共享
面向对象设计 线程是主体 任务是主体，符合单一职责
扩展性 受限于单继承 可实现多个接口
推荐场景 简单场景，快速实现 复杂场景，推荐使用

方式一：继承Thread类

实现步骤

  1. 创建一个类继承自Thread
  2. 重写run()方法，定义线程执行体
  3. 创建线程对象并调用start()方法启动

完整代码

/**
 * 通过继承Thread类创建线程
 */
public class MyThread extends Thread {
    
    private String threadName;
    
    public MyThread(String threadName) {
        this.threadName = threadName;
    }
    
    @Override
    public void run() {
        for (int i = 1; i <= 5; i++) {
            System.out.println(threadName + " 执行第 " + i + " 次");
            try {
                // 模拟耗时操作
                Thread.sleep(100);
            } catch (InterruptedException e) {
                Thread.currentThread().interrupt();
                System.out.println(threadName + " 被中断");
            }
        }
        System.out.println(threadName + " 执行完毕");
    }
    
    public static void main(String[] args) {
        System.out.println("=== 继承Thread类方式 ===");
        
        // 创建线程对象
        MyThread thread1 = new MyThread("线程A");
        MyThread thread2 = new MyThread("线程B");
        
        // 启动线程
        thread1.start();
        thread2.start();
        
        System.out.println("主线程继续执行...");
    }
}

方式二：实现Runnable接口

实现步骤

  1. 创建一个类实现Runnable接口
  2. 实现run()方法，定义任务逻辑
  3. 创建Thread对象，传入Runnable实例
  4. 调用Thread的start()方法启动

完整代码

/**
 * 通过实现Runnable接口创建线程
 */
public class MyRunnable implements Runnable {
    
    private String taskName;
    
    public MyRunnable(String taskName) {
        this.taskName = taskName;
    }
    
    @Override
    public void run() {
        for (int i = 1; i <= 5; i++) {
            System.out.println(taskName + " 执行第 " + i + " 次");
            try {
                Thread.sleep(100);
            } catch (InterruptedException e) {
                Thread.currentThread().interrupt();
                System.out.println(taskName + " 被中断");
            }
        }
        System.out.println(taskName + " 执行完毕");
    }
    
    public static void main(String[] args) {
        System.out.println("\n=== 实现Runnable接口方式 ===");
        
        // 创建任务对象
        MyRunnable task1 = new MyRunnable("任务A");
        MyRunnable task2 = new MyRunnable("任务B");
        
        // 创建线程，传入任务
        Thread thread1 = new Thread(task1);
        Thread thread2 = new Thread(task2);
        
        // 启动线程
        thread1.start();
        thread2.start();
        
        System.out.println("主线程继续执行...");
    }
}

方式三：实现Callable接口（带返回值）

实现步骤

  1. 创建一个类实现Callable接口
  2. 实现call()方法，定义任务逻辑并返回结果
  3. 通过FutureTask包装Callable
  4. 创建Thread对象，传入FutureTask
  5. 调用start()启动，通过get()获取结果

完整代码

import java.util.concurrent.Callable;
import java.util.concurrent.ExecutionException;
import java.util.concurrent.FutureTask;

/**
 * 通过实现Callable接口创建线程（带返回值）
 */
public class MyCallable implements Callable<Integer> {
    
    private String taskName;
    
    public MyCallable(String taskName) {
        this.taskName = taskName;
    }
    
    @Override
    public Integer call() throws Exception {
        int sum = 0;
        for (int i = 1; i <= 5; i++) {
            sum += i;
            System.out.println(taskName + " 累加: " + i + ", 当前和: " + sum);
            Thread.sleep(100);
        }
        System.out.println(taskName + " 执行完毕，结果: " + sum);
        return sum;
    }
    
    public static void main(String[] args) {
        System.out.println("\n=== 实现Callable接口方式（带返回值） ===");
        
        // 创建Callable任务
        MyCallable callable1 = new MyCallable("计算任务A");
        MyCallable callable2 = new MyCallable("计算任务B");
        
        // 包装为FutureTask
        FutureTask<Integer> futureTask1 = new FutureTask<>(callable1);
        FutureTask<Integer> futureTask2 = new FutureTask<>(callable2);
        
        // 创建线程并启动
        Thread thread1 = new Thread(futureTask1);
        Thread thread2 = new Thread(futureTask2);
        
        thread1.start();
        thread2.start();
        
        // 获取返回结果
        try {
            Integer result1 = futureTask1.get();
            Integer result2 = futureTask2.get();
            System.out.println("\n任务A结果: " + result1);
            System.out.println("任务B结果: " + result2);
            System.out.println("总和: " + (result1 + result2));
        } catch (InterruptedException | ExecutionException e) {
            e.printStackTrace();
        }
        
        System.out.println("主线程执行完毕");
    }
}

// 使用lambda创建线程
public class LambdaThreadDemo {

        // Runnable方式
        new Thread(() -> {
            for (int i = 0; i < 5; i++) {
                System.out.println("Lambda线程: " + i);
