四、Java反射机制

发表于2026-06-23

题目描述

使用Java反射机制获取类的信息并动态调用方法。

解题思路

  1. 通过Class.forName()获取类对象
  2. 使用反射获取类的构造器、方法、字段
  3. 通过反射创建对象实例
  4. 动态调用对象的方法

完整代码

import java.lang.reflect.Constructor;
import java.lang.reflect.Field;
import java.lang.reflect.Method;

public class ReflectionDemo {
    
    public static class User {
        private String name;
        private int age;
        public String email;
        
        public User() { this.name = "Unknown"; this.age = 0; }
        public User(String name, int age) { this.name = name; this.age = age; }
        
        public String getName() { return name; }
        public void setName(String name) { this.name = name; }
        public int getAge() { return age; }
        public void setAge(int age) { this.age = age; }
        
        public void sayHello(String greeting) {
            System.out.println(greeting + ", 我是 " + name);
        }
        
        private void secretMethod() {
            System.out.println("这是一个私有方法");
        }
    }
    
    public static void printClassInfo(Class<?> clazz) {
        System.out.println("类名: " + clazz.getName());
        System.out.println("\n=== 构造函数 ===");
        for (Constructor<?> c : clazz.getDeclaredConstructors()) {
            System.out.println("构造函数: " + c);
        }
        System.out.println("\n=== 方法列表 ===");
        for (Method m : clazz.getDeclaredMethods()) {
            System.out.println("方法: " + m.getName());
        }
    }
    
    public static void main(String[] args) throws Exception {
        Class<User> clazz = User.class;
        printClassInfo(clazz);
        
        // 创建对象
        User user = clazz.getDeclaredConstructor(String.class, int.class).newInstance("张三", 25);
        
        // 调用方法
        Method sayHelloMethod = clazz.getDeclaredMethod("sayHello", String.class);
        sayHelloMethod.invoke(user, "你好");
        
        // 访问私有字段
        Field nameField = clazz.getDeclaredField("name");
        nameField.setAccessible(true);
        System.out.println("获取私有字段name: " + nameField.get(user));
        
        // 调用私有方法
        Method secretMethod = clazz.getDeclaredMethod("secretMethod");
        secretMethod.setAccessible(true);
        secretMethod.invoke(user);
    }
}

知识点总结

  • Class对象：类的元数据，可以通过三种方式获取
  • 反射的用途：框架开发、动态代理、依赖注入
  • setAccessible(true)：突破访问限制，访问私有成员
