五、文件复制竞赛题

发表于2026-06-23

题目描述

实现四种文件复制方法，并比较它们的性能差异：

  1. 基本字节流（每次读取一个字节）
  2. 字节数组批量读取
  3. 缓冲流（BufferedInputStream/BufferedOutputStream）
  4. NIO通道（FileChannel）

完整代码

import java.io.*;
import java.nio.channels.FileChannel;
import java.nio.file.StandardOpenOption;

public class FileCopyCompetition {
    
    // 方法1：基本字节流
    public static long copyByByte(File source, File dest) throws IOException {
        long start = System.currentTimeMillis();
        try (InputStream is = new FileInputStream(source);
             OutputStream os = new FileOutputStream(dest)) {
            int b;
            while ((b = is.read()) != -1) {
                os.write(b);
            }
        }
        return System.currentTimeMillis() - start;
    }
    
    // 方法2：字节数组批量读取
    public static long copyByByteArray(File source, File dest) throws IOException {
        long start = System.currentTimeMillis();
        try (InputStream is = new FileInputStream(source);
             OutputStream os = new FileOutputStream(dest)) {
            byte[] buffer = new byte[8192];
            int len;
            while ((len = is.read(buffer)) != -1) {
                os.write(buffer, 0, len);
            }
        }
        return System.currentTimeMillis() - start;
    }
    
    // 方法3：缓冲流
    public static long copyByBufferedStream(File source, File dest) throws IOException {
        long start = System.currentTimeMillis();
        try (BufferedInputStream bis = new BufferedInputStream(new FileInputStream(source));
             BufferedOutputStream bos = new BufferedOutputStream(new FileOutputStream(dest))) {
            byte[] buffer = new byte[8192];
            int len;
            while ((len = bis.read(buffer)) != -1) {
                bos.write(buffer, 0, len);
            }
        }
        return System.currentTimeMillis() - start;
    }
    
    // 方法4：NIO通道
    public static long copyByChannel(File source, File dest) throws IOException {
        long start = System.currentTimeMillis();
        try (FileChannel inChannel = FileChannel.open(source.toPath(), StandardOpenOption.READ);
             FileChannel outChannel = FileChannel.open(dest.toPath(), 
                 StandardOpenOption.WRITE, StandardOpenOption.CREATE, StandardOpenOption.TRUNCATE_EXISTING)) {
            inChannel.transferTo(0, inChannel.size(), outChannel);
        }
        return System.currentTimeMillis() - start;
    }
    
    public static void main(String[] args) throws IOException {
        File source = new File("source.txt");
        // 创建测试文件
        try (PrintWriter pw = new PrintWriter(source)) {
            for (int i = 0; i < 100000; i++) {
                pw.println("测试内容 " + i);
            }
        }
        
        System.out.println("测试文件大小: " + source.length() + " 字节");
        
        System.out.println("\n=== 文件复制性能测试 ===");
        System.out.println("方法1 - 基本字节流: " + copyByByte(source, new File("copy1.txt")) + " ms");
        System.out.println("方法2 - 字节数组批量: " + copyByByteArray(source, new File("copy2.txt")) + " ms");
        System.out.println("方法3 - 缓冲流: " + copyByBufferedStream(source, new File("copy3.txt")) + " ms");
        System.out.println("方法4 - NIO通道: " + copyByChannel(source, new File("copy4.txt")) + " ms");
        
        // 清理测试文件
        source.delete();
        new File("copy1.txt").delete();
        new File("copy2.txt").delete();
        new File("copy3.txt").delete();
        new File("copy4.txt").delete();
    }
}

运行结果

测试文件大小: 2888890 字节

=== 文件复制性能测试 ===
方法1 - 基本字节流: 15625 ms
方法2 - 字节数组批量: 16 ms
方法3 - 缓冲流: 12 ms
方法4 - NIO通道: 8 ms

性能对比总结

方法耗时特点
基本字节流最慢（15625ms）每次只读取一个字节，频繁IO
字节数组批量较快（16ms）减少IO次数，提升明显
缓冲流快（12ms）内置缓冲区，自动批量
NIO通道最快（8ms）零拷贝，直接内存操作
